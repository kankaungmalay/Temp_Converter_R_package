# A Temperature Converter R package

A random practice for creating personal use R packages.

## Usage:

In R command line,

```r
library(devtools)

install_github("kankaungmalay/Temp_Converter_R_package")

library(TempConverter)

F_to_C(45)

C_to_F(24)
```
